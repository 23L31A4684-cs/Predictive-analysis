This project demonstrates predictive analytics using historical sales data to forecast future trends.
The main objective of this project is to analyze past data and predict future outcomes using machine learning techniques.

The dataset used in this project contains sales values recorded over a period of time.
The data is preprocessed by converting categorical values such as month names into numerical format.
This preprocessing step helps the machine learning model to understand the data properly.

A Linear Regression algorithm is used to build the predictive model.
The model learns the relationship between the time (months) and sales values.
After training, the model is used to predict future sales for upcoming months.

The performance of the model is evaluated using accuracy metrics such as R-squared score.
This helps in understanding how well the model fits the data.

Data visualization is performed using graphs to represent actual data, regression line, and predicted values.
The graph helps in clearly understanding trends and future predictions.

 # Prograaming language used
The project is implemented using Python programming language.
Pandas library is used for data handling and preprocessing.
NumPy is used for numerical operations.
Matplotlib is used for plotting graphs and visualization.
Scikit-learn is used for building and training the Linear Regression model.

# run the project
To run this project, first install Python on your system.
Open Command Prompt or Terminal.
Install the required libraries using the command: pip install pandas numpy matplotlib scikit-learn
Navigate to the project folder using the cd command.
Go to the src folder where the Python file is located.
Run the program using the command: python model.py

# output of the programm
After running the program, the model will train on the dataset.
Future predictions will be displayed in the terminal.
A graph will be generated showing actual and predicted values.
The graph will also be saved in the output folder.
